# notadez
ao utilizar NotaDez, o docente cadastrará uma ou mais instituições e turmas . Importará para o sistema os estudantes ou poderá cadastrá-los um a um vinculados a uma turma. Na sequência natural de uso, poderá apontar notas em diversas atividades, provas e possibilitará o cálculo de nota final da disciplina de maneira automática.
